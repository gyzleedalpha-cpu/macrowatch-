
MACROWATCH — Instrucciones de instalación
==========================================

PASO 1 — Instalar Python
-------------------------
1. Andá a https://www.python.org/downloads/
2. Descargá Python 3.x (el botón amarillo grande)
3. Al instalar, MARCA la opción "Add Python to PATH"
4. Completá la instalación


PASO 2 — Obtener tu API Key de Anthropic
-----------------------------------------
1. Andá a https://console.anthropic.com
2. Creá una cuenta gratuita
3. Andá a "API Keys" → "Create Key"
4. Copiá la clave (empieza con sk-ant-...)


PASO 3 — Pegar tu API Key
--------------------------
1. Abrí el archivo server.py con el Bloc de notas
2. En la línea 4 reemplazá:
   API_KEY = "TU_API_KEY_AQUI"
   Por tu clave real, por ejemplo:
   API_KEY = "sk-ant-api03-xxxxxxxxxxxx"
3. Guardá el archivo


PASO 4 — Ejecutar
------------------
Opción A (fácil): Doble click en INICIAR.bat
Opción B (manual): Abrí CMD en esta carpeta y escribí:
   python server.py

El navegador se abre solo en http://localhost:8080


NOTAS
------
- El servidor solo funciona mientras la ventana CMD esté abierta
- Para usarlo desde el celular (misma red WiFi):
  Reemplazá "localhost" por la IP de tu PC, ej: http://192.168.1.X:8080
- Costo aproximado: ~$0.002 por actualización (muy barato)
